## APO 1-meter Telescope

Working with Dr. Thomas Harrison at my undergraduate institution,
New Mexico State University (NMSU), I wrote the
software to control a high speed 5-band photometer on the 1-meter NMSU
telescope at the Apache Point Observatory.
The software was written in LabView and provided an easy to use graphical user
interface that controlled the data
acquisition and telescope positioning. The image was taken from this
[link](http://www.apo.nmsu.edu/09siteA.jpg).

## 2.5-meter Sloan Telescope

During the summer before my graduate courses started, I undertook and additional
software design project at NMSU
with Dr. Jon Holtzman. My role was to create a program that would allow
observers to use the Apache Point
Observatory Galactic Evolution Experiment (APOGEE) spectrograph, located on the
2.5 meter Sloan telescope, from the
adjacent NMSU 1-meter telescope. This allows observers to collect spectra of
astronomical objects during bright time
(near full moon) on a smaller aperture telescope, when the larger aperture Sloan
telescope cannot be used. The
software package I wrote for this project was written in Python.
